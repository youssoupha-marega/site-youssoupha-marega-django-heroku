docs

But
- Fournir la documentation et les notes internes du projet : procédures, instructions d'ajout d'articles, création d'apps, astuces.

Contenu important
- `add_articles.txt` : procédure pour ajouter des articles.
- `creer_app.txt` : guide pour créer une nouvelle app Django dans le projet.
- `utils.txt` : notes utilitaires diverses.

Usage
- Destiné aux contributeurs et pour l'onboarding.

Bonnes pratiques
- Transformer les notes importantes en fichiers `md` structurés si elles sont souvent consultées.
- Ajouter des checklist avant déploiement (migrations, collectstatic, vérification des variables d'environnement).
